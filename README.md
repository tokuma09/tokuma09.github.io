## Overview

* Blog posts are hosted in `content/blog`


### Creating new content

Run this command from the root of project

```bash
hugo new path/to/new-content.md
```

### Local Check

```bash
hugo server -D
```

## Updating the theme

```bash
git submodule update --remote
```
